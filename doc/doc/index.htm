What is Miranda ICQ?

Miranda ICQ is an ICQ clone for Microsoft Windows. Unlike the offical ICQ for Windows, Miranda is small, fast and easy.

Shortcuts

Miranda ICQ has a few shortcuts that will work no matter what application your using (as long as miranda is running):
Ctrl + Shift + A - Show/Hide the main Miranda Window
Ctrl + Shift + I - Show an unread messages
Ctrl + Shift + S - Search the Web


Note: These keys are customizable through the registry. This is only for advanced users. Later versions will allow customization through the options window

Status Icons

Status icons are pictures you see that indicate your and other peoples status (ie Offline, Online, NA etc).

Miranda comes with a basic set of icons, but you can change these. Miranda will look for a Icons.dll on startup, if it exists it will use this to get its status icons. See the Miranda Source code for help making your own Icon Dlls.

Profiles

Miranda ICQ Supports Profiles, this allows you to have more than one ICQ account on one machine. By default the profile used is default. If you wish to use a different profile, just run Miranda with the commandline option of the profile name.
E.g. Miranda32.exe Bob This will run miranda and use the profile Bob, if the profile Bob does not exist then it will be created.

For details about files and how they work with profiles see the section Storage

Storage

Each profile will have 2 or 3 files. If the registry is used to store settings then there will only be two files.

profile.cnt - All your contacts
profile.hst - Your history
profile.ini - Only if you don't use the Registry, stores all your settings.

Note: profile represents your profile name (see Profiles).
